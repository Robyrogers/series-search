# series-search

It's a simple React Web App that uses the open movie database api to search for movies and shows.

A pet project to learn more about how React works.

Requirements:
>Node.js
>npm

Steps to run locally:
> Download or clone the repo to your machine
> Run npm install through the command line on the repo directory
> After the first time npm install is done you can just run npm start to run the app every time in the local directory through the command line.
